Overview

Conventional metronomes and practice tools are often too simplistic. BeatMatrix allows you to create and customise any rhythm and time signature. You can create a grid (or bar) of up to twenty beats. Each of these can be subdivided and accents can be applied. They can also be linked together to create any complex rhythm structure. Additionally, BeatMatrix allows speed to be set over the course of the grid to model accelerando and ritardando. Beat patterns can be saved within the app.

BeatMatrix is highly recommended for musicians working with contemporary scores, Indian music (tāla), Balkan and Celtic additive metres, modern jazz polyrhythms, or situations where there are odd and complex time signatures.

BeatMatrix was created to let musicians construct, chain, and repeat tempo-defined rhythmic units, mirroring the way time is structured in the music itself. Musicians will find this tool useful in helping to visualise, hear and feel intricate temporal structures. An extremely useful feature is the linking of grids to create the effect of multiple metronomes and the ability to set speed changes with a customisable transition curve.

Features Summary

  • Customisable beat patterns
  • Add accents, subdivisions and repeats
  • Link patterns together
  • Reorder patterns at any time
  • Set speed changes
  • Set tempo transition curve
  • Save patterns
