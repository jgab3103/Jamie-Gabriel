Aim: Provide definitions of standard metrics

Definition: Integration Metrics measure the connections between nodes. These metrics include:

  1. Definition: The Distance between two nodes is the number of edges that can be traversed to move from one to node to the other node. Also called the Path
  2. Definition: A Shortest Path is the minimum number of edges that can connect two nodes.
  3. Definition: The Diameter is then number of edges contained in the longest Shortest Path
  4. Definition: The Characteristic Path Length is the average of all Shortest Paths within the graph. It can be calculated by using the formula $ \frac{1}{q(q - 1)} \Sigma_{i \in{V}} l_i $, where $V$ is set of nodes and $q = |V|$
  5. Definition: The Global Efficiency is the inverse of the Shortest path for all pairs of nodes, and is a measure of how efficiently information can be moved across a network. It is given by the formula $ \frac{1}{q(q - 1)} \Sigma_{i \in V} \frac{1}{l_{ij}}$
  6. Definition: The Local Effiency is calculated in the same manner of Global Efficiency, but considers only the neighborhood of the node, not the node itself

Definition: Segregation Metrics measure the presence of groups of interconnected nodes (or communities) within a network. These metrics include:

  1. Definition: The Clustering Coefficient measures how much nodes cluster together, defined as a fraction of triangles, which is the fraction of a node's neighbors that are neighbors of each other. These is also a local version of this metric called the Local Clustering Coefficient that considers each node in a network.
  2. Definition: The Transitivity of a graph is the ratio between observed number of closed triplets (i.e. complete subgraphs with 3 nodes and 2 edges) and maximum possible closed triplets.
  3. Definition: The Modularity of a network quantifies the dvision of a network into groups (which may be called modules, clusters, groups, sub-graphs) and measures which groups are more closed that others.

Definition: Centrality Metrics assess the importance of individual nodes inside a given network. These metrics include:

  1. Defintion The Degree centrality measures the number of incident edges on a given node $i$. This can be further defined by in-degree centrality and out-degree centrality referring to the direction of the edges
  2. Definition: The Closeness Centrality quantifies how much a node is connected to other nodes. It is the average of distance of node $i$ to all other nodes. If $l_{ij}$ is the shortest path between $i$ and $j$, closeness centrality is defined as: $ \frac{1}{\Sigma_{i \in V, i\ne j} l{ij}}$
  3. Definition: The Betweeness Centrality metric evaluates how much a node acts as a bridge between nodes, given by the formula $ \Sigma_{w \ne i i \ne j} \frac{L_{wj}(i)}{L_{wj}} $ where $L_{wj}$ is total number of shortest paths between node $w$ and node $j$ and $L_{wj}(i)$ is the total number of paths between $w$ and $j$ passing through $i$

Definiton: Resiliance Metrics measure how much a network can maintain and adapt its operational performance when facing failure. These metrics include:

  1. Definition: The Assortativity quantifies the tendency of nodes being connected to similiar nodes and there are different measures that are used to calculate this.
  2. Definition: The Pearson Correlation Coefficient measures the assortativity between degrees of directly connected nodes. It assumes postive values when there is a correlation between nodes of a similiar degree
