Overview

The purpose of this project is to capture and analyse MIDI data in real time and use it to enhance live improvised performance. The data includes both note-on/note-off events as well as continuous control changes generated by MIDI-enabled effects and instruments.

All data is recorded, timestamped, and analysed using real-time deep learning models. These models generate new MIDI data which can be fed back to performers as musical stimulus during improvisation.

The goal is not to automate performance or simulate human improvisers. Instead, the system enhances the improvisational environment by allowing performers to interact with latent musical structures as they emerge — a real-time feedback loop between musician, data, and sound.

This is a multidisciplinary project combining improvisation practice, live sound design, computational modelling, and symbolic music representation. The system becomes a responsive participant in the musical dialogue.

Finally, the name Deep Improv is a nod to Deep Listening, the lifelong project of Pauline Oliveros, extending that lineage into real-time MIDI and spectrogram-based improvisation.
